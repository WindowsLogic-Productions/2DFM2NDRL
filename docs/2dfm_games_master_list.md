# 2DFM Games — Master List

Combined catalog of every 2D Fighter Maker (FM95 / FM2K / KGT2K) game tracked
across:

- **Local collection**: `/mnt/c/games/2dfm/`
- **Wiki master list**: <https://2dfm.miraheze.org/wiki/Games_Master_List>
- **User-contributed entries** (extra blurbs / genre info / titles missing
  from both sources above)

`On disk?` is `Yes` if a folder for the game exists under `/mnt/c/games/2dfm/`.
